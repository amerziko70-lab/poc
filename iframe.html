Iframe (من نفس URL اللي أعطيتَه)

هذا الـ iframe يحمل الصفحة الخارجية لكن لا يسمح لها بالوصول إلى صفحة الأم (إزالة allow-same-origin لأسباب أمان).

بديل للاختبار الآمن — تواصل عبر postMessage (موصى به)

لو تريد اختبار تواصل بين الإطار والصفحة الأم بدون تسريب حسّاس، استخدم postMessage. المثال التالي يُنشئ iframe inline (srcdoc) يرسل رسالة عند التحميل، والصفحة الأم تستقبلها:

// parent listens
window.addEventListener('message', function(ev){
  console.log('Received message from iframe origin:', ev.origin);
  console.log('Data:', ev.data);
  // هنا نعرض رسالة آمنة فقط — لا نحاول قراءة أو إرسال cookies
});
